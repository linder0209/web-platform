Como JS提供了对full版和only版的切换支持：

full版：对JS原生对象如String, Function, Array, Date, Object以及常用的Class和$变量进行了扩展，如可以直接使用" abc ".trim(); $("#test");

only版：没有对JS原生对象、Class, Object, $进行扩展，所有对象函数都属于Como对象，故只能使用Como.String.trim(' abc '); Como("#test");
在使用String, Function, Array, Date等这些对象时，函数的第一个函数为String等这些对象，如Como.Function.bind(function(){}, this, arg1, arg2);

默认为full版，若您的网页中有其他JS框架，请选择only版；

如何定义版本，在Como JS的src地址上加上"?only"即可，如

<script type="text/javascript" src="/js/Como.js?only"></script>
