Cookie
Como.Cookie对象
get set del
get
取Cookie值
Como.Cookie.get(name);
set
设置Cookie值
Como.Cookie.set(name, value);
Como.Cookie.set(name, value, expires);		//expires=0过期时间为永不过期，expires=1为1分钟，expires = null ? '' 过期时间为浏览器关闭时间
Como.Cookie.set(name, value, expires, path, domain);
del
删除Cookie
Como.Cookie.del(name, value, domain);
