Como.extend
扩展Como里面的属性和方法
Como.extend({
	getMyName : function(){
		return "kevin";
	}
});
var name = Como.getMyName();
var name1 = $.getMyName();
Como.template
模板机制(v1.1.1更新)；除了String.format提供简单模板实现外，这里又提供一个基于Key的模板机制, 通过{key}来匹配;
方法有两个： set(key, value)或者set(object), run()取得最后模板的值， (value支持任意Object), 如例子
var test = Como.template("my name is {name}, i am \\{sex\\}, i am {old} years old!");
test.set('name', 'kevin').set('old', 25).set('sex', 'body');
//这里也可以test.set({'name':'kevin', 'old': 25});
//为什么sex没有出来，因为通过\\将匹配符过滤掉了
log(test.run());
